# react-typescript-101

This is the simplest example I can get for TypeScript and React.

Technologies used:

- React 16
- TypeScript 2.7
- Webpack 4

For Testing:
- Jest
- Enzyme


## Run the project

`npm install`

To install the project

`npm run build`

To Build the bundle.js

Open index.html in your browser.

Test the project

`npm test`
